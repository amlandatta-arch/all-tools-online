üîç How to Test Mobile View

Method 1: Browser DevTools (Recommended)

  1. Open index.html in Chrome, Firefox, or Edge
  2. Press F12 to open DevTools
  3. Press Ctrl+Shift+M (or click device icon) to toggle device toolbar
  4. Select a device:
    • iPhone SE - 375x667
    • iPhone 12 Pro - 390x844
    • Samsung Galaxy S20 - 360x800
    • iPad - 768x1024
  5. Refresh the page (F5)
  6. Test all features listed below

Method 2: Resize Browser Window

  1. Open index.html in your browser
  2. Drag the browser window edge to make it narrower
  3. Watch how the layout adapts at different widths

Method 3: Test on Real Device

  1. If you have a local server running, access via phone/tablet
  2. Or deploy to a hosting service and test the live URL
